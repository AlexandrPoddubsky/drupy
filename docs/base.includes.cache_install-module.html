Module cache_install

source code

Stub of cache functions for use during installation A stub cache implementation to be used during the installation process when database access is not yet available. Because Drupal's caching system never requires that cached data be present, these stub functions can short-circuit the process and sidestep the need for any persistent storage. Obviously, using this cache implementation during normal operations would have a negative impact on performance.
